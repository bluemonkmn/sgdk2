Information

The source code included in this archive uses Microsoft's .NET platform. The .NET framework has a built-in C# compiler. You do not need to download any SDK or development platform. Simply run makesgdk2.bat to compile SGDK2IDE.exe. You DO need to have the Microsoft .NET framework installed in order to compile and in order to run SGDK2IDE.exe. The framework can be downloaded from Microsoft's download page

You will also need DirectX 9.0, and you might need to modify makesgdk2.bat, depending on the version of DirectX and the version of the .NET framework you are using. The output executable is named SGDK2IDE.exe. This file is included in the archive for your convenience in case you are unable to configure a build environment.

Also included in this archive is a test project that shows off a few of the feratures implemented so far. TestPrj.sg2 can be loaded into the program and viewed to see how SGDK2 will handle overlapping and composite tiles.
